# 遊戲玩法：
### 開始遊戲：
###### 使用 `CMD` 指令進入遊戲。
### 遊戲操控：
###### w - 往上；
###### a - 往左；
###### s - 往下；
###### d - 往右；
### 遊戲規則：
###### 每移動一次，就會產生四個數字格。數字格最大為4，最小為0 (等於不生成)。
###### 遊戲沒有結束的機制。
